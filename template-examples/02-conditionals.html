{% endif %} {% if is_admin %}

How it works:

Define condition variables:
<!-- template:var is_logged_in="1" is_admin="0" -->

Basic if statement:
{% if is_logged_in %}
  User is logged in.
{% endif %}

If-else statement:
{% if is_admin %}
  User is an admin.
{% else %}
  User is not an admin.
{% endif %}

Truthy and Falsy Values

The following values are considered "true":

  • "1", "true", "yes", "y", "on" and any non-empty string not explicitly false

The following values are considered "false":

  • "0", "false", "no", "n", "off" and empty strings

Examples:

    {% if has_subscription %}
  • has_subscription (value: "{{has_subscription}}") is TRUE
    • {% else %}
  • has_subscription (value: "{{has_subscription}}") is FALSE
    • {% endif %} {% if status %}
  • status (value: "{{status}}") is TRUE
    • {% else %}
  • status (value: "{{status}}") is FALSE
    • {% endif %} {% if empty_var %}
  • empty_var (value: "{{empty_var}}") is TRUE
    • {% else %}
  • empty_var (value: "{{empty_var}}") is FALSE
    • {% endif %}

Nested Conditions

{% if is_logged_in %}

User is logged in.

{% if is_admin %}

User has admin privileges.

{% else %}

User has regular privileges.

{% if has_subscription %}

User has an active subscription.

{% endif %} {% endif %} {% else %}

User is not logged in.

{% endif %}

How it works:

You can nest conditions inside other conditions:
{% if is_logged_in %}
  User is logged in.
  {% if is_admin %}
    User has admin privileges.
  {% else %}
    User has regular privileges.
  {% endif %}
{% endif %}
