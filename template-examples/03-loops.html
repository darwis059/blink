Basic Loop

Fruit List:

    {% for item in items %}
  • {{item}}
    • {% endfor %}

How it works:

Define loop items in an HTML comment:
<!-- template:items "Apple" "Banana" "Cherry" "Date" "Elderberry" -->

Basic loop syntax:
{% for item in items %}
  <li>{{item}}</li>
{% endfor %}

Loop with Index

Numbered List:

    {% for item in items %}
  1. {{item}}
    2. {% endfor %}

Note: The template engine doesn't directly support loop indices, but you can use ordered lists to achieve numbered items.

Loops with Multi-part Items

Product List:

{% for item in items %} {% endfor %}
Name Category Color Price
{{item.0}} {{item.1}} {{item.2}} ${{item.3}}

How it works:

Define items with pipe-delimited fields:
<!-- template:items "Apple|fruit|red|0.99" "Carrot|vegetable|orange|1.25" ... -->

Access individual parts with dot notation:
<td>{{item.0}}</td>
<td>{{item.1}}</td>
<td>{{item.2}}</td>
<td>${{item.3}}</td>

Nested Loops

Categorized List:

{% for item in items %}

{{item.0}}:

  • {{item.1}}
{% endfor %}

Note: The template engine doesn't support string splitting, but in a more advanced implementation, you could have nested loops iterating through comma-separated values.
